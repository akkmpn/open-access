# Daily Auto-Reset Todo Checklist - Setup Guide

## Features
✅ Check tasks during the day
✅ Automatically unchecks at midnight (local time)
✅ Tracks completion percentage
✅ Shows daily streak
✅ Stores data locally in browser
✅ Share link in Telegram group

## How to Use

### Adding Tasks
1. Type task name in input field
2. Select priority (High/Medium/Low)
3. Click "Add Task"

### Daily Workflow
1. Check off tasks as you complete them during the day
2. See real-time completion % at top
3. At midnight, all checkboxes automatically reset
4. Next day, repeat the same tasks

### Manual Reset
- Click "Reset Today" button to manually reset all tasks
- Useful if you want to start fresh mid-day

### Viewing Streak
- "Streak" shows how many consecutive days you've used the app
- Completes automatically each midnight

### Accessing from Telegram
- Save this HTML file
- Upload to any free hosting (see options below)
- Share the link in your Telegram group
- Anyone can access and check their tasks

## Free Hosting Options to Share Link

1. **GitHub Pages** (Best - Free forever)
   - Upload HTML file to GitHub
   - Enable GitHub Pages
   - Get permanent shareable link

2. **Netlify** (Easy - Free tier)
   - Drag & drop HTML file
   - Auto-generates shareable link
   - No account needed

3. **Vercel** (Fast - Free tier)
   - Connect GitHub or upload file
   - Instant shareable URL

4. **Firebase Hosting** (Google - Free)
   - Free tier with generous limits
   - Shareable public URL

5. **OneDrive/Google Drive** (Simple - Already have it)
   - Upload HTML file
   - Share link (opens in browser)
   - Everyone can access

## Best Practice for Telegram Group
- Share the link with description: "Daily Todo Checklist - Opens in browser, no account needed"
- Pin the link in your Telegram group for easy access
- Group members can each use it independently

## Customization
- Edit the HTML file to add your own default tasks
- Change colors in CSS section
- Adjust reset time (currently midnight local time)
